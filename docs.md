## ❓ Why use this? / Зачем это нужно?

### 🇬🇧 English
* **No token wasting**: Reduce token billing by up to 65% with prompt compression.
* **100% Secure**: Sandbox dangerous agent CLI operations inside Docker containers.
* **Open Source & Free**: Avoid expensive subscriptions with free LLM keys and custom OpenAI compatibility layers.

### 🇷🇺 Русский
* **Экономия токенов**: Сократите расходы на генерацию длинных ответов ИИ до 65% с помощью сжатия.
* **Безопасность**: Запускайте агентов в изолированном Docker-контейнере, чтобы защитить личные данные на хосте.
* **Халява и гибкость**: Настройте совместимые роутеры подписки (ClinePass) и используйте бесплатные API-ключи.
